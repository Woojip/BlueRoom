## Local Development

1. Clone the repository
2. Serve with any local HTTP server:
   ```powershell
   # Using PowerShell script
   ./start-server.ps1
   ```
3. Open http://localhost:8080

## Quest 2 Usage

1. Ensure PC and Quest 2 are on the same network
2. Open Quest 2 browser
3. Navigate to the live URL or your local IP
4. Click "Enter VR" button

## Technologies

- [A-Frame](https://aframe.io/) - WebXR Framework
